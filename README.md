## 🚀 Getting Started

### Prerequisites

- Node.js (v14+)
- npm or yarn

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/yourusername/portfolio.git
   ```

2. Install packages

   ```sh
   npm install
   # or
   yarn
   ```

3. Run the development server

   ```sh
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and visit `http://localhost:5173/portf/`
